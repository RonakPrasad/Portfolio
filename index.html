Legato Guitar Store

Guitar Stores are musical instruments store which sell only guitars and guitar-related equipment. The goal of this project is to provide customers the provision to view guitars and buy them online through a website. There are two databases used, one for the users’data and the other for the store's data. The website is user-interactive.

  • The users must create an account to buy products or add them to their shopping cart.
  • The user can add and remove items from their shopping cart and this data is stored or deleted from the database based on the user's activity.
  • The user can place the order by filling the shipment and billing details.
  • This data is stored in the database for the store to deliver the order.
  • The user can track the order even without logging in.
  • This is how this project helps in showing how an actual store can work on the internet.

Technologies Used:

HTML, CSS, JavaScript, PHP, MySQL

Euphorica

Euphorica is a two-in-one Android Application. The two components of the app are: Firstly, Musical Note Detector/Tuner and the other a Metronome, both of them have different functionality.

  • The Musical Note Detector or Tuner recognizes a musical note when played by a musical instrument.
  • It is known as a Tuner because it is used to tune untuned instruments.
  • The Metronome is a practice tool that produces a steady beat to help musicians play rhythms accurately.
  • Both of these tools are essential for building your musicianship and are stepping stones for playing music accurately.

Technologies Used:

Android Studio, Java, XML, Apache Commons Math3 Library

Visual Novel

A Visual Novel is an interactive storyline based game genre (although it’s not considered a game) which involves text-based story and narration along with static or character-based visuals, graphics and sounds.

  • It is basically an interactive book with visual artworks and sound effects in which a reader(user) can make choices and based on those choices the story can have different storyline and multiple endings.
  • The fact that visual novels are distinguished from other game types by their generally minimal gameplay is still a controversy.
  • Typically the majority of player interaction is limited to clicking to keep the text, graphics and sound moving, while making narrative choices along the way.
  • Another main characteristic of visual novels are its strong emphasis on the story, as the narration in visual novels are delivered through text.
